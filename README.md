This project is a static coffee shop landing page ("Bean & Brew") with no database requirements. The site consists of six sections: Hero, About, Menu, Hours, Contact, and Footer. All content is statically rendered with no dynamic data layer. Contact form submissions are logged to the console only and not persisted. No Supabase tables, storage, or authentication are used. The `db/schema.sql` and `db/seed.sql` files are placeholders to satisfy project structure requirements but contain no operational schema.
